# ChatGPT: Reasoning Analysis Template

## Purpose

Perform deep, step-by-step reasoning analysis for complex problems.

## Template

```
Analyze this problem step-by-step:

Problem: [PROBLEM]
Context: [CONTEXT]
Constraints: [CONSTRAINTS]

Use chain-of-thought reasoning:
1. Understand the problem
2. Break down into sub-problems
3. Analyze each sub-problem
4. Synthesize conclusions
5. Validate against constraints

Provide actionable recommendations.
```

## Variables

- `[PROBLEM]` - Problem statement
- `[CONTEXT]` - Additional context (product, market, etc.)
- `[CONSTRAINTS]` - Constraints (budget, time, resources, etc.)

## Example

```
Analyze this problem step-by-step:

Problem: How to build a data moat for a design system SaaS?
Context: 
  - Product: Enterprise Design System for startups
  - Market: Startups, scale-ups, CTOs
  - Current state: No data moat yet
Constraints:
  - Limited resources (solo founder)
  - Fast time-to-market needed
  - Must be defensible long-term

Use chain-of-thought reasoning:
1. Understand the problem
2. Break down into sub-problems
3. Analyze each sub-problem
4. Synthesize conclusions
5. Validate against constraints

Provide actionable recommendations.
```

## Tips

1. **Be specific** - Clear problem statement = better analysis
2. **Include context** - More context = more relevant analysis
3. **List constraints** - Constraints guide recommendations
4. **Request step-by-step** - Chain-of-thought improves quality
