# Midjourney: Hero Image Template

## Purpose

Generate hero images for landing pages and marketing materials.

## Template

```
/imagine prompt: [PRODUCT_CONCEPT], [STYLE], [MOOD], [COMPOSITION], --ar [ASPECT_RATIO] --style raw --v 6
```

## Variables

- `[PRODUCT_CONCEPT]` - What the product does/represents
- `[STYLE]` - Visual style (minimalist, modern, professional, etc.)
- `[MOOD]` - Emotional mood (optimistic, professional, energetic, etc.)
- `[COMPOSITION]` - Layout/composition (centered, grid, etc.)
- `[ASPECT_RATIO]` - Aspect ratio (16:9, 1:1, 9:16, etc.)

## Examples

### SaaS Dashboard

```
/imagine prompt: Modern SaaS dashboard interface, clean minimalist design, professional tech startup aesthetic, bright optimistic mood, centered composition, --ar 16:9 --style raw --v 6
```

### Design System

```
/imagine prompt: Design system component library, organized grid layout, professional UI design, clean workspace mood, top-down view, --ar 16:9 --style raw --v 6
```

### Abstract Brand

```
/imagine prompt: Abstract geometric shapes representing design system, modern minimalist style, professional brand aesthetic, balanced composition, --ar 1:1 --style raw --v 6
```

## Tips

1. **Be specific** - More detail = better results
2. **Use style modifiers** - `--style raw` for realistic, `--v 6` for latest
3. **Test variations** - Generate multiple, select best
4. **Aspect ratio matters** - Match intended use (16:9 for web, 1:1 for social)
