# ElevenLabs: Demo Narration Template

## Purpose

Create professional voice narration for product demos.

## Template

```
Create demo narration script:

Product: [PRODUCT]
Demo flow: [FLOW]
Key features: [FEATURES]
Call-to-action: [CTA]

Script should:
1. Welcome viewers
2. Introduce product value
3. Highlight key features
4. End with clear CTA

Keep script under [DURATION] seconds.
Optimize for natural voice narration with pauses.
```

## Variables

- `[PRODUCT]` - Product name
- `[FLOW]` - Demo flow/structure
- `[FEATURES]` - Key features to highlight
- `[CTA]` - Call-to-action
- `[DURATION]` - Target duration in seconds

## Example

```
Create demo narration script:

Product: Enterprise Design System
Demo flow:
  1. Welcome and problem statement
  2. Show component library
  3. Demonstrate collaboration features
  4. Show analytics dashboard
  5. Call to action
Key features:
  - Pre-built component library
  - Team collaboration
  - Design system analytics
Call-to-action: Start your free trial today

Script should:
1. Welcome viewers
2. Introduce product value
3. Highlight key features
4. End with clear CTA

Keep script under 90 seconds.
Optimize for natural voice narration with pauses.
```

## Tips

1. **Keep it short** - 60-90 seconds is ideal
2. **Natural pauses** - Include commas and periods
3. **Clear CTA** - End with specific action
4. **Feature focus** - Highlight 2-3 key features max
